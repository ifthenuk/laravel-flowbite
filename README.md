# Laravel with Flowbite Admin Dashboard

Laravel A fully responsive admin dashboard template built with Tailwind CSS and Flowbite.

## Features

* Laravel Breeze
* Fully responsive and mobile-friendly
* Built with Tailwind CSS and Flowbite
* Easy to customize and extend
* Includes a range of pre-built UI components
* Supports dark mode and light mode

## Technology Stack

* Laravel: PHP framework for web development
* Tailwind CSS: Utility-first CSS framework for styling
* Flowbite: Low-code UI component library for Tailwind CSS

## Installation

1. Clone the repository: `https://github.com/ifthenuk/laravel-flowbite.git`
2. Install php dependecies: `composer install`
3. Install the dependencies: `npm install`
4. Build the project: `npm run build`
5. Start the development server: `composer run dev`
